# StackPilot Profile Templates

These templates are starter profile exports for VS Code-compatible editors.

The folder includes single-stack profiles, a Markdown knowledge-base profile, and a first batch of combined full-stack profiles for monorepos such as `Python + Vue` or `Java + React`.

To use one:

1. Open the Command Palette.
2. Run `Profiles: Import Profile...`.
3. Select a `.code-profile` file from this folder.
4. Review the contents and create the profile.
5. Use the profile for the current workspace.

StackPilot does not install or remove extensions automatically. The templates are recommendations that users can review before applying.
